# Constants

## What I Learned
Constants are values that do not change. Use "const" keyword. there are two types constant. e.g. Typed constants : they are declared with a defined type.
     Untyped constants : declared without a type.
You can also group constant together in a block for readability called : multiple constants declaration.    

## Example
```go
const PI = 3.14

const A int = 1 : Type constant
const A = : Untype constant
const (
    A int = 1
    B = 2.14
    c = "Hi"
)  : Multiple constant declaration
