# Go Data Types

## What i learnt
Go data type specify the type of variables value. It has basic types: int, float, string, bool. It also supports arrays, slices, maps, structs.

## Example
```go
var age int = 25, default value = 0
var name string = "Joy", default value = ""(empty string)
var bool boolean = true, default value = false
var dec float32 = 3.14, default value = 0.0
