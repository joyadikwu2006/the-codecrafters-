# go comments

## what i learnt

comment in go are used to explain the code and when executing the code it is been ignored, comments can be in single-line or multi-line, it can be used to prevent code when executing an alternative code.

# Example
```go
** package main
import "fmt"
func main() {
    // this is a single line comment
    fmt.Println("state a comment")
    /* ignore these it is a multi-line comment do not execute it */
}
