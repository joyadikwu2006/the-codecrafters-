# operator

## What i learnt 
operator perform operation on variables and values. 

**there are type of operator**
* Arithmetic operator: +, /, *, %, ++, --, -.
there are used for mathematical operation.

* Assignment operator: =, +=, <<=, >>=, -=, /=, %=, |=, &=, e.t.c
there are used to assign values to variables.

* Comparison operator: ==, !=, <=, =>, <, >.
there are used to compare two values.

* Logical operator: &&, !, ||.
used to determine the logic between variables or value.

* Bitwise operator: >>, ^, |, &, <<.
there are for binary numbers.
